Bootstrap is a free and open-source front-end framework for designing websites and web applications. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

To use Bootstrap in your project, you can include the Bootstrap CSS and JavaScript files in your HTML file. You can either use a CDN link or download the files from the Bootstrap website and host them locally.
